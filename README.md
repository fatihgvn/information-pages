# Development

**Install NodeJs**
> https://nodejs.org/en/download/
```
npm install
```

**Install Grunt**
> https://gruntjs.com/getting-started
```
npm install -g grunt-cli
```
or with sudo
```
sudo npm install -g grunt-cli
```

**Run server**
```
grunt run
```

**Build With Minify**
```
grunt build
```
